subway: 至简，致心

Aug 02, 2024

至简，致心。

一个极简而克制的 Hexo 主题，灵感来源于 Metro UI 。这里是 Github 链接。

此站点目前正在使用此主题。

为什么写这个主题

我一直都很喜欢 Windows 8 ，主要原因之一就是它使用的 Metro UI 。那种简洁而纯粹的风格正是我一直想要的。然后最近有了拥有自己的博客主题的想法，于是，就有了这个主题。

这也是我内心某种纯洁而朴素的情感的表达。

关于设计

目前我并没有完全复刻 Windows 8 开始屏幕的观感，而是使用了 Metro UI 中最引人注目的一个元素：大字报标题。以后如果有机会，我会试着做出开始屏幕那样的效果；不过目前学业繁重，能抽出时间做到这个程度已经是万幸。

页面上并没有使用任何色彩，也没有什么夺人眼球的动画。好的博客应该专注于内容本身，而不是各式各样的视觉效果。

目前没有支持 tags, categories 或者 archive 的计划，可能是因为我个人不怎么用这些功能吧。以后可能会加上的。

最后

随缘更新吧。欢迎提 issue ，欢迎 PR 和 fork 。
